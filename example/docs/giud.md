# 简介
------
lite 是一款基于 Vue.js 2.5 的轻量级前端 UI 组件库，主要用于快速开发 PC 网站中后台，界面简约，代码健壮。

### 诞生背景
之前一直使用ElementUI，公司最近来了些定制项目，UI设计已成型的那种，许多element组实现效果并不好，重写了大量组件，所以萌生了自己写一套UI样式，以应付各种定制需求

### 特性
 * 基于Vue 2.5
 * Semantic UI 风格样式
 * 友好的 API

### 浏览器支持
* 现代浏览器和 IE9 及以上

### 警告

lite现阶段还没有发布正式版本，可以阅读学习，但是并不适合用于生产
